# 使用`keras` 实现简单的图片分类任务

## 0.准备计算环境

需要安装`tensorflow`  `keras` 等

## 1. 准备数据

### 下载
从这里下载： https://www.kaggle.com/c/dogs-vs-cats/data

### `COPY`
然后分别copy图片到如下目录下（需要自己创建目录）

```
data/
    train/
        dogs/
            dog001.jpg
            dog002.jpg
            ...
        cats/
            cat001.jpg
            cat002.jpg
            ...
    validation/
        dogs/
            dog001.jpg
            dog002.jpg
            ...
        cats/
            cat001.jpg
            cat002.jpg
            ...
```
